About pktana

pktana is a modern Linux network inspection toolkit built for engineers, SREs, security analysts, and infrastructure teams. It unifies packet capture, protocol analysis, NIC diagnostics, routing inspection, connection tracking, and dataplane visibility into a single production-grade executable.

Instead of juggling multiple tools like tcpdump, ethtool, ss, ip route, and iftop, pktana delivers a consistent, high-performance interface designed for modern cloud-native infrastructure.

Why Engineers Choose pktana

Deep Packet Inspection

Full L2-L7 protocol decoding including TCP, UDP, TLS, DNS, HTTP, VXLAN, Geneve, BGP, and more.

Advanced TUI Dashboard

Real-time connections, process tracking, GeoIP, filtering, sorting, and historic flow analysis.

Dataplane Intelligence

Detect XDP, AF_XDP, DPDK, SR-IOV, hardware offloads, queue mappings, and IRQ affinity.

PCAP Export

Capture live traffic and export standard PCAP files for offline Wireshark analysis.

Zero External Dependencies

Single native binary with minimal runtime requirements. Built entirely in Rust.

Production Ready

Designed for data centers, cloud platforms, network security, and carrier-grade infrastructure.

Quick Start

# Launch interactive dashboard sudo pktana tui eth0 # Capture packets sudo pktana capture eth0 # Display active connections pktana conn # Inspect NIC hardware pktana ethtool eth0 # Analyze dataplane pktana dp eth0 # Query GeoIP pktana geoip 8.8.8.8
