About pktana

pktana is a next-generation Linux network analysis platform engineered for network developers, SREs, security engineers, and infrastructure teams. It delivers packet capture, full L2–L7 protocol decoding, flow analysis, NIC introspection, routing visibility, and dataplane intelligence in one unified tool.

Replace fragmented workflows built around tcpdump, ethtool, ss, iproute2, Wireshark, and multiple ad-hoc scripts with a single high-performance Rust application — distributed as a signed RPM — designed for production environments.

Why Engineers Choose pktana

🔬 Deep Packet Inspection

Full L2–L7 decode: Ethernet, IPv4/IPv6, TCP full options, UDP, ICMP, TLS, QUIC, HTTP/2, gRPC, WebSocket, SSH, SIP, NTP, BGP, DNS, DHCP, VXLAN, GRE inner frames, and 20+ more protocols.

🔑 TLS JA3 Fingerprinting

Full ClientHello parsing — cipher suites, ALPN negotiation, elliptic curves, GREASE filtering. JA3 raw string ready for threat intel lookup. TLS 1.0/1.1 deprecation warnings inline.

⚡ QUIC & HTTP/2

QUIC long/short header decode with version identification (RFC 9000, RFC 9369, gQUIC). HTTP/2 frame parsing with gRPC path extraction. WebSocket frame opcode decode.

🛡️ Risk Scoring

Composite 0–100 risk score aggregating: deprecated TLS, SSHv1, NTP amplification, DNS entropy spikes (DGA/tunneling), NULL scan, zero-window SYN, and tunnel anomalies.

🌍 Tunnel Inner-Frame Decode

VXLAN, GRE, and Geneve tunnels are re-inspected — inner IPv4 src/dst, proto, ports, and application protocol are decoded from the encapsulated frame.

📊 DNS Entropy Analysis

Shannon entropy computed on the longest DNS label per query. High entropy (>3.8 bits) flags potential DGA domain generation or DNS tunneling activity.

🖥️ Wireshark-Like TUI

5-tab live dashboard: Overview, Packets, Flows, Stats, Help. Detail popup with Original / Layers / Hex sub-tabs. All DPI fields — JA3, QUIC, tunnel, risk bar, app category — displayed in real time.

🌐 GeoIP Everywhere

Offline IP-to-country lookup in pktana conn (per connection), pktana stats (top talkers), and pktana geoip. No API keys. No internet required.

⚙️ Dataplane Analytics

Detect XDP eBPF programs, AF_XDP zero-copy sockets, DPDK/vfio-pci bindings, SR-IOV VF/PF topology, per-queue IRQ affinity, NUMA node, PCIe link width/speed, and hardware offloads.

🔗 Active Connections

Color-coded TCP state (ESTABLISHED, LISTEN, TIME_WAIT), GeoIP country per remote IP, service name resolution, PID → process name — all from procfs without root for most info.

📦 Zero Dependencies

Single native Rust binary. libpcap is the only runtime dependency (auto-installed by the RPM). No Python, no Java, no Node.js, no extra packages.

🏭 Production Grade

Built for cloud platforms, telecom networks, data centers, and carrier-grade infrastructure. Clean clippy (-D warnings), memory safe, no unsafe code in the parsing path.

Quick Start

# Wireshark-like TUI dashboard sudo pktana tui eth0 # Live capture — color-coded + DPI-enriched Info column sudo pktana eth0 sudo pktana eth0 'port 443' # BPF filter # Deep packet inspection — all L2–L7 fields, risk score, JA3 pktana inspect 450000280001400040060000c0a80001... # Active connections — GeoIP + state colors + service names pktana conn # Live stats dashboard — top talkers with GeoIP country sudo pktana stats eth0 # NIC hardware / offloads / IRQ affinity pktana ethtool eth0 # Dataplane: XDP / DPDK / SR-IOV / AF_XDP pktana dp eth0 # Routing table (IPv4 + IPv6) pktana route # Offline GeoIP lookup (no API key needed) pktana geoip 8.8.8.8 1.1.1.1 9.9.9.9
