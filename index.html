About pktana

pktana is a next-generation Linux network analysis platform engineered for network developers, SREs, security engineers, and infrastructure teams. It delivers packet capture, flow analysis, protocol decoding, NIC introspection, routing visibility, and dataplane intelligence in one unified tool.

Replace fragmented workflows built around tcpdump, ethtool, ss, iproute2, and multiple ad-hoc scripts with a single high-performance Rust application designed for production environments.

Install Instantly

Optimized for RHEL, Rocky Linux, AlmaLinux, and CentOS Stream.

sudo dnf install \ https://github.com/omnayak27199/pktana/releases/latest/download/pktana-0.3.0-1.el9.x86_64.rpm

Why Engineers Choose pktana

Deep Packet Inspection

Layer 2 through Layer 7 protocol decoding including TCP, UDP, TLS, DNS, HTTP, VXLAN, Geneve, and BGP.

Interactive TUI

Real-time connections, process correlation, GeoIP, advanced filtering, and historical flow visibility.

Dataplane Analytics

Detect XDP, AF_XDP, DPDK, SR-IOV, RSS queues, IRQ affinity, and hardware offload capabilities.

PCAP Export

Capture live traffic and export industry-standard PCAP files for Wireshark and forensic analysis.

Zero Dependencies

Single native Rust binary with minimal runtime requirements and exceptional portability.

Production Grade

Built for cloud platforms, telecom networks, data centers, and carrier-grade infrastructure.

Quick Start

# Launch interactive dashboard sudo pktana tui eth0 # Capture packets sudo pktana capture eth0 # Display connections pktana conn # Inspect NIC hardware pktana ethtool eth0 # Analyze dataplane pktana dp eth0 # GeoIP lookup pktana geoip 8.8.8.8
