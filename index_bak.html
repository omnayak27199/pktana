🎉 What's New in v0.3.0

🔍 Process Tracking

Maps network connections to process names and PIDs by scanning /proc. See which processes own each connection in the TUI dashboard.

💾 PCAP Export

Save captured packets to PCAP files for offline analysis with Wireshark. Full libpcap savefile support with proper timestamps.

🚀 Advanced TUI Dashboard

Sortable columns, real-time filtering, mouse support, historic connections tracking, and multi-tab interface with keyboard shortcuts.

🌍 GeoIP Integration

Offline IP geolocation using MaxMind GeoLite2 database. Country codes displayed for all remote addresses in the TUI.

🎮 Enhanced Controls

Press s to sort, / to filter, t to toggle historic, Tab to switch tabs. Full mouse support with click and scroll.

📊 Traffic Statistics

Per-protocol breakdown, connection tracking, byte/packet counters, and traffic rate calculations in real-time.

🚀 Quick Start

# Start TUI dashboard (requires root for packet capture) sudo pktana tui eth0 # Capture packets to file sudo pktana capture eth0 --count 100 # Query GeoIP database pktana geoip 8.8.8.8 # Show NIC hardware details (ethtool equivalent) pktana ethtool eth0 # Deep packet inspection from hex pktana inspect "4500003c..."

✨ Core Features

📊 TUI Dashboard

Real-time terminal UI with connection tracking, process mapping, and GeoIP lookups.

📦 Packet Capture

Live capture with BPF filters, DNS decode, and colorized output.

🔬 Deep Inspection

Layer-by-layer packet decode: Ethernet, ARP, IPv4, TCP/UDP/ICMP, TLS, HTTP, DNS.

🤖 Auto-Diagnosis

Rule-based findings: TCP handshake state, zero-window, OS fingerprints, anomaly detection.

🔧 NIC Information

Interface details from sysfs: state, MAC, MTU, speed, IP addresses, RX/TX counters.

⚡ ethtool Equivalent

Driver info, PCI bus, firmware version, link speed, IRQ affinities, queue stats.
